==============
website baidu map
==============

联系我们网页上的地图替换成百度地图

Installation
============

正常安装即可

Configuration
=============

请按以下步骤进行设置:

#. 在website admin菜单里，填入您的百度地图AK。如果没有注册，点击旁边的连接，会打开注册页面。

#. 设置公司的经度、纬度。如果你不知道你的公司位置的坐标，点击旁边的连接，会打开百度地图坐标选取页面。

Usage
=====

设置好之后，打开odoo网站，在联系我们页面，你会发现原来的谷歌地图已经替换为百度地图了。
